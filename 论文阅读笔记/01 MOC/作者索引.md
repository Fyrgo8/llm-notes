# 作者索引

## 说明

这个页面基于 `02 Papers/` 里的 frontmatter `authors` 字段工作。

因此它现在对下面这类笔记最有效：

- 新工作流生成的论文笔记
- 已补齐 `authors:` frontmatter 的旧笔记

如果某篇旧笔记只在正文里写了“作者：...”，但没有写进 frontmatter，这里不会自动识别。

## 所有带作者元信息的论文

```dataview
TABLE authors, year, venue, status, topics
FROM "02 Papers"
WHERE authors
SORT year DESC, file.name ASC
```

## 按作者展开浏览

```dataview
TABLE year, venue, status, topics
FROM "02 Papers"
WHERE contains(authors, "Dong, Guanting")
SORT year DESC
```

## 后续建议

- 以后新生成的论文笔记默认都写 `authors:` frontmatter
- 旧笔记如果值得长期维护，再逐步补齐 frontmatter
- 一旦 `02 Papers/` 里的笔记都统一结构，这个页面就会越来越有用
