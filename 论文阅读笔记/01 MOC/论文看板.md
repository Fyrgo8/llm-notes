# 论文看板

## 最近加入的论文

```dataview
TABLE authors, year, venue, status, file.mtime AS updated
FROM "02 Papers"
SORT file.mtime DESC
LIMIT 20
```

## 按状态查看

```dataview
TABLE authors, venue, year, topics
FROM "02 Papers"
SORT status ASC, file.name ASC
GROUP BY status
```

## 按主题查看

```dataview
TABLE authors, year, venue, status
FROM "02 Papers"
SORT year DESC, file.name ASC
GROUP BY topics
```

## 按作者查看

```dataview
TABLE authors, year, venue, status, topics
FROM "02 Papers"
WHERE authors
SORT year DESC, file.name ASC
```

## 最近深读完成

```dataview
TABLE authors, year, venue, topics, updated
FROM "02 Papers"
WHERE status = "deeply-read"
SORT updated DESC
```

## 待补元数据

```dataview
TABLE authors, year, venue, zotero_key
FROM "02 Papers"
WHERE !year OR length(authors) = 0 OR !zotero_key
SORT file.name ASC
```
